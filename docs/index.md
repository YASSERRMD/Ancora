# Ancora

**Local-first, privacy-respecting multi-agent runtime.**

Ancora lets you build, orchestrate, and operate AI agents that run wherever your
data lives -- on-device, in your datacenter, or air-gapped. No data leaves your
infrastructure unless you explicitly configure a remote provider.

## Why Ancora?

| Property | Ancora |
|----------|--------|
| **Local-first** | Runs on a laptop or edge node with a local model |
| **Durable** | Journal-backed replay survives crashes and restarts |
| **Deterministic** | Idempotent activity recording prevents double-sends |
| **Multi-language** | Go, Python, TypeScript, .NET, Java, Rust |
| **Privacy by design** | Data stays in your perimeter by default |
