# Ancora v0.6.0: Ecosystem Milestone

The Ancora ecosystem is now fully consolidated and production-ready.

## What is new

Plugin catalog, registry, sample applications, interop toolkit, and full test coverage are all green.

## Breaking changes

- `PluginCtx::invoke` has been renamed to `PluginCtx::call`
- Plugin manifests must use schema v3

## Getting started

```bash
cargo add ancora
```

Follow the extension author quickstart guide at [ancora.dev/docs/quickstart](https://ancora.dev/docs/quickstart).
