# Marketplace Trust

The Ancora marketplace surfaces a trust score for each plugin to help users make informed decisions.

## Trust score components

| Component        | Max contribution |
|------------------|-----------------|
| Download count   | 30%             |
| Review score     | 30%             |
| Security audit   | 30%             |
| Active maintenance | 10%           |

## Featured plugins

A plugin is eligible for the featured section when its trust score is 0.7 or higher
and its security audit status is `Passed`.

## Improving your trust score

- Submit your plugin for a community security audit.
- Respond promptly to user reviews and issues.
- Keep your dependencies up to date.
